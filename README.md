# Drones_Setup
This is the repo for TUT Drone's Setup Files

1. Folder Drone_Frame: link to solidworks files
2. Folder Firmware_config_files: files that are needed for Arducopter 3.3.x for first configuration
3. Pixhawk and Remote Radio Configuration.pdf - includes most important configuration guidelines
4. Pixhawk with 3.3.2 firmware.pdf - includes most important aspects for tuning the firmware and config. the drone
